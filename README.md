An expansive vore mod for Crusader Kings 3 - Heavy Work in Progress

Currently this includes a new safe-vore themed religion, which has a vore-themed event, a prison vore system, and the abilty to vore lovers.

There is no artwork yet, so missing texture icons are expected

**Install Instructions**

Download the folder and drop it in the C:\Users\%USER%\Documents\Paradox Interactive\Crusader Kings III\mod folder.
