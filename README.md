An expansive vore mod for Crusader Kings 3 - Heavy Work in Progress

* Oral Vore
* Anal Vore
* Unbirth
* Cock Vore
* Female Preds
* Male Preds
* Disposal
  
Currently this includes a new safe-vore themed religion, which has a vore-themed event, a prison vore system, some random events, and the abilty to vore lovers.

There is no artwork yet, so missing texture icons are expected

Credit to Bloodborn for inspiring this mod

**Install Instructions**

1. Download the folder and drop it in the C:\Users\%USER%\Documents\Paradox Interactive\Crusader Kings III\mod folder.
2. Cut the Consumption.mod file and paste it directly into the mod folder

**Incompatibilities**

Incompatible with mods that change window_character.gui. If you want to patch it, more the prey_list_button to your current window_character.gui
Should be compatible with everything else
