Currently this includes a new safe-vore themed religion, which has a vore-themed event, a prison vore system, some random events, and the abilty to vore lovers

Credit to Bloodborn for inspiring this mod

**Install Instructions**

1. Download the folder and drop it in the C:\Users\%USER%\Documents\Paradox Interactive\Crusader Kings III\mod folder
2. Enable it in the Paradox Launcher
3. If you are enabling it in an old save, make sure to use the vore toggles decision for it to work properly

**Incompatibilities**

Incompatible with mods that change window_character.gui. If you want to patch it, move the prey_list_button from the mod's window_character.gui to your current window_character.gui and place it just above the barber button.
Incompatible with mods that change the character's body, including CBO. You can mitigate this by changing the game/gfx/portraits/female_body/female_body.asset to female_body.asset.bak
